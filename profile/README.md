# What is Quarterly Learnings?

Quarterly Learnings is an independent technology education and consulting shop specializing in web-based technology solutions. Examples include in-person web development classes, custom CMS implementations, and virtual software & AI upskilling with top universities and Fortune 100s.

## What is this GitHub organization for?

This GitHub organization houses the repos I have used to share knowledge and facilitate learning for myself and others. So far, it has helped me, my nephew, my neighbor, dozens of grant recipients, and hundreds of students of all ages.

I hope that it helps you.

It also houses other projects that support teaching assignments, workshop facilitation, or freelance/contract development.
